Time complexity

T(n) = aT(n/2) + 1 for n>1 and T(n) = T(1) for n=1
T(n) = O(n)

T(n) = aT(n/2) + n for n>1 and T(n) = T(1) for n=1
T(n) = O(nlog₂n)

T(n) = aT(√n) + 1 for n>1 and T(n) = T(1) for n=1
n = 2ᵐ
T(n) = O(log₂n)

T(n) = aT(√n) + log₂n for n>1 and T(n) = T(1) for n=1
n = 2ᵐ
T(n) = O(log₂n log₂log₂n)
